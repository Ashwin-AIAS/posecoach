# data/ — Datasets, Eval Results & Thesis Exports

## What This Directory Is
All generated data. Nothing in this directory is hand-written — it's all produced
by scripts in `scripts/`. Most of it is gitignored (large files).

## Directory Map
```
data/
├── kaggle/          # Raw Kaggle dataset — GITIGNORED (large videos)
├── keypoints/       # Extracted keypoints as .npy files — GITIGNORED
├── chroma/          # ChromaDB vector store — GITIGNORED (regenerate with python -m app.chatbot.ingest)
├── eval/            # Thesis evaluation results — GITIGNORED (regenerated by eval scripts)
│   ├── yolo_results.json
│   ├── latency_results.json
│   ├── consistency_results.json
│   ├── chatbot_results.json
│   └── sus_responses/        # One JSON per participant
└── thesis/          # Final outputs for thesis document — GITIGNORED
    ├── *.csv                  # Tables for thesis
    ├── *.png                  # Figures for thesis
    └── latex/                 # LaTeX snippets for copy-paste into thesis
        ├── table_accuracy.tex
        ├── table_latency.tex
        └── figure_score_distribution.tex
```

## What Is and Is NOT Gitignored
```gitignore
data/kaggle/       ← gitignored (videos are too large)
data/keypoints/    ← gitignored (regenerate with extract_keypoints.py)
data/chroma/       ← gitignored (regenerate with python -m app.chatbot.ingest)
data/eval/         ← gitignored (regenerate with eval scripts)
data/thesis/       ← gitignored (regenerate with export_thesis_tables.py)
```

## Regenerating Data
```bash
# Regenerate ChromaDB (after adding/editing knowledge files)
python -m app.chatbot.ingest

# Regenerate all eval results
python scripts/eval_yolo.py
python scripts/eval_latency.py
python scripts/eval_form_consistency.py
python scripts/eval_chatbot.py
python scripts/eval_user_study.py

# Regenerate thesis exports
python scripts/export_thesis_tables.py
```

## Privacy Rule
JPEG frames are NEVER stored in this directory or anywhere on disk.
The only image data allowed is PNG figures generated by eval scripts for the thesis.
Keypoint .npy files are allowed (they contain coordinates, not pixel data).
